For an MVP, I'd keep it to around **12–15 screens**.

| Screen                                     | Main Content                                                                                   |
| ------------------------------------------ | ---------------------------------------------------------------------------------------------- |
| **1. Login**                               | Email/Employee ID, Password, Forgot Password                                                   |
| **2. Employee Dashboard**                  | Current review status, pending actions, previous reviews, notifications                        |
| **3. My Performance Reviews**              | List of review cycles (2026 Mid-Year, Annual, etc.), status                                    |
| **4. Performance Review Form**             | Objectives (1–10), weight (optional), self-score (1–10), comments/evidence, Save Draft, Submit |
| **5. Review Summary**                      | Total score, average, comments, submission confirmation                                        |
| **6. Line Manager Dashboard**              | Team overview, pending approvals, completed reviews, department statistics                     |
| **7. Manager Review Screen**               | Employee details, self-scores, manager scores, comments, approve/return for correction         |
| **8. HR Dashboard**                        | Reviews awaiting HR, department progress, completion percentage, overdue reviews               |
| **9. HR Review Screen**                    | Employee review, manager comments, HR comments, approve/reject, final score                    |
| **10. MD Dashboard**                       | Overall company performance, department averages, top performers, low performers, charts       |
| **11. Department Performance**             | Department ranking, average scores, employee list, drill-down into individuals                 |
| **12. Employee Profile**                   | Personal details, department, manager, review history, performance trend                       |
| **13. Review Cycle Management (Admin/HR)** | Create review cycle, start/end dates, departments, objectives, status                          |
| **14. Objective Management**               | Create/edit objectives by department, weights, scoring rules                                   |
| **15. User & Role Management**             | Employees, managers, HR, MD, departments, permissions                                          |

### Dashboards by Role

**Employee**

* Current review
* Review history
* Notifications
* Manager feedback

**Line Manager**

* Pending approvals
* Team completion %
* Average team score
* Employees needing attention

**HR**

* Company completion rate
* Department progress
* Pending approvals
* Reports & exports

**MD**

* Company-wide KPIs
* Department comparisons
* Performance distribution
* Top/Bottom performers
* Trends across review cycles

This structure is scalable—you can later add features like **360° feedback**, **goal setting (OKRs/KPIs)**, **training recommendations**, and **promotion/salary review workflows** without redesigning the core system.
